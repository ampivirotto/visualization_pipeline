FAQs

What if I encounter an error in the conversion of GTC files from IDAT files?

Verify to make sure the mainfiest and cluster files are correct (CSV, BPM, EGT) based on the Beadchip type used in the study. This can be found on the GEO page for the series with the prefix GPL.

What if I lose a lot of variants during the GTC2VCF or normalization process?

Again verify that the manifest and cluster files are correct. Additionally check to make sure the correct Reference fasta sequence is being used.
